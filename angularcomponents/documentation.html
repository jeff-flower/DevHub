vbutton component

link
 link is the url or path to the link
displaytext
 display text is the text displayed on the button

Example
<vbutton link="http://www.google.com" displaytext="Example"> </vbutton>

vprogress component

percentage
  number representing percentage complete

Example
<vprogress percentage=50> </vprogress>

vicon component

testsuite
 testsuite as a string

Example
<vicon testsuite="Restaurant Bundle"> </vicon>

vicons component

testsuites
 array of test suite strings

Example
<vicons testsuites='["Restaurant bundle", "Retail bundle", "Gift", "Ecommerce bundle (ODP)", "ODP Ecommerce", "Ecommerce DevHub"]'> </vicons>

panel component

app
 json object with app info

Example 
        <panel  app=
                            
            '{
                "name" : "TestApp2",
                "description" : "description for test app 2",
                "percentage" : 20,
                "suites" : [ "ecommerce", "gift", "restaurant"]
            }'> 
        </panel>

appheader component

Input: N/A

 Will generate the static "app header" content

Example
<appheader> </appheader>

tabs component

Attributes/Headers

tabinfo
json array defining the tabs and what view they show
  • id
    The id of the div to show when the tab is selected
  • text
    Display text of the tab
  • active
    Whether the tab should be active by default

content
 array of application data

Example 
        <tabs 
            tabinfo=
            '
                [{"id" : "inprogress", "text" : "In Progress", active: "is-active"},
                {"id" : "completed", "text" : "Completed", active: ""}]
            ' 
            content=
            '
                {
                  "inprogress" : [
                      {
                         "name" : "Dino-Fighter",
                         "description" : "Description for Dino-Fighter app",
                         "percentage" : 10,
                         "suites" : [ "ecommerce", "gift"]
                      }, 
                      {
                         "name" : "TestApp2",
                         "description" : "description for test app 2",
                         "percentage" : 20,
                         "suites" : [ "ecommerce", "gift", "restaurant"]
                      }
                    ],
                  "completed" : [
                      {
                         "name" : "TestApp3",
                         "description" : "description for test app 3",
                         "percentage" : 100,
                         "suites" : [ "ecommerce", "gift"]
                      }, {
                          
                         "name" : "Unicorns Suck",
                         "description" : "Description for unicorns suck",
                         "percentage" : 100,
                         "suites" : [ "ecommerce", "restaurant"]
                      }
                    ]
                }
            ' 
        >
        </tabs>
